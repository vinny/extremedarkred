# ExtremeDark Red

<img src="https://i.imgur.com/qRmeVeq.png" alt="ExtremeDark Red" width="720">

ExtremeDark Red is a dark red phpBB style based on prosilver.

This version is being developed for phpBB 4.0.0 alpha 2, with the visual identity inspired by the older ExtremeDark Red releases for phpBB 3.0.x

## Information

- Style name: ExtremeDark Red
- Author: Vinny
- Based on: prosilver
- phpBB version: 4.0.0 alpha2

## Notes

This style is currently under development and should be tested before use on a production board.
